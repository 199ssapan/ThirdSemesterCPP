Реализовать Паттерн Фабрика на языке C++
Фабрика должна производить объекты для записи конфигураций

Типы конфигураций:
1) YAML (https://github.com/jbeder/yaml-cpp)
2) JSON ( https://github.com/open-source-parsers/jsoncpp)
3) Plain Text

объекты должны удовлетворять общему интерфейсу
IConfigWriter
Методы интерфейса:
- Write Key
- Read Key
- Write File
- Read File

Для демонстрации собрать объект для каждого типа конфигурации, записать в него ключ, сохранить файл, считать файл в новый объект и прочитать ключ
