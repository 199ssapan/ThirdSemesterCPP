реализовать программу по учету оценок за диплом
программа должна иметь базу данных студентов
у студента есть имя, фамилия, отчество, оценка за диплом,
также пользователю должна предоставляться возможность сортировать студентов по всем вышеперечисленным полям,
для сортировки использовать std::sort с переданной лямбда функцией в качестве критерия сортировки
